# 抖音视频下载器

极简的抖音视频无水印下载工具

## 功能特性

- 🎯 极简界面设计
- 📱 支持抖音分享链接解析
- 🚫 无水印高清视频下载
- 🎬 视频预览功能
- 📱 响应式设计，支持移动端

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动服务
```bash
npm start
```

### 开发模式
```bash
npm run dev
```

访问 http://localhost:3000 开始使用

## 使用说明

1. 在抖音APP中分享视频，复制链接
2. 粘贴链接到输入框
3. 点击"解析视频"按钮
4. 预览或直接下载视频

## 技术栈

- 前端：HTML + CSS + 原生JavaScript
- 后端：Node.js + Express
- 网络请求：axios
- 跨域处理：cors
